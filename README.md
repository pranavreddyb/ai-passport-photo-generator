# AI Passport Photo Generator

AI-powered web application that converts ordinary portraits into professional passport-ready photos using background removal, face detection, intelligent cropping, and image enhancement.

---

## Features

- AI background removal using rembg
- Automatic face detection with OpenCV
- Smart head-and-shoulders cropping
- Brightness, contrast, and sharpness enhancement
- High-resolution passport photo generation
- Premium SaaS-style web interface
- Live before-and-after preview
- One-click photo download

---
